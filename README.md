## 0. architecture:

### 0-1. user aspect:

user - react(front-end) - spring boot(back-end)

### 0-2. developer aspect:

github(CI) - codebuild(AWS - CD) - [ S3(AWS - front Storage), EC2(AWS - Server) ]- CloudFront(AWS - CDN) - route53(AWS - DNS, hositng)

## 1. environment:
- node v18.17.0
- yarn 1.22.19
- react 18.2.0
- spring boot 3.0.9

## 2. how to Run
download node.js, yarn

$ yarn install

$ yarn start
